# Module: HTML & Semantic Structure

## Description
HTML is the skeleton of every webpage. In this module, you will learn how to structure content using semantic tags. Semantic HTML makes your code cleaner, easier for browsers to understand, and accessible to everyone.

## Learning Objectives
- Use common HTML5 tags (h1-h6, p, ul, ol, li).
- Understand the importance of Semantic HTML (header, main, section, footer, article).
- Link external resources like images and stylesheets.
