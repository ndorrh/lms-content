# Project: Portfolio Milestone 1 - The Skeleton

### Objective
Create the initial HTML structure for your personal portfolio. Focus ONLY on content and semantics—don't worry about colors or fonts yet!

### Milestone Goals
- Create an `index.html` file.
- Use a `<header>` for your name and navigation.
- Use a `<main>` section for your bio.
- Use a `<footer>` for copyright info.

### Instructions
1. Create a folder named `portfolio`.
2. Inside, create `index.html`.
3. Build your page structure using:
   - An `<h1>` for your name.
   - An `<h2>` for "About Me".
   - A `<p>` block for your short bio (4-6 sentences).
   - An `<img>` tag (use a placeholder if you don't have a photo yet).
4. Validate your HTML!

Submit your GitHub repository link. If you wish to include a screenshot, please add it to your repository.
