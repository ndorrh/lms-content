# Lesson: HTML Fundamentals & Semantic Tags

### Overview
Writing HTML is about describing the *meaning* of your content, not just its appearance. 

### Key Concepts
- **Structural Tags**: `<div>` vs `<section>`.
- **Text Tags**: `<h1>`, `<p>`, `<strong>`, `<em>`.
- **Lists**: Ordered (`<ol>`) and Unordered (`<ul>`).
- **Semantic HTML**: Tags that carry meaning (e.g., `<nav>`, `<article>`, `<header>`).
