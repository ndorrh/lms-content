# Lesson: Git Workflow & GitHub

### Overview
Git is like a "Save Game" system for your code. It allows you to take snapshots (commits) of your progress. GitHub is where those snapshots live in the cloud.

### Key Concepts
- **Repository (Repo)**: A folder tracked by Git.
- **Commit**: A snapshot of your code at a specific time.
- **Push**: Sending your local commits to GitHub.
- **README**: The manual for your repository.
