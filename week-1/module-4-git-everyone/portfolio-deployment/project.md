# Project: Portfolio Milestone 3 - Go LIVE!

### Objective
Host your personal portfolio on the web so that anyone can visit it.

### Milestone Goals
- Create a GitHub repository named `tech_ascend_portfolio`.
- Push your `portfolio` code to this repository.
- Deploy your site using GitHub Pages or Vercel.

### Instructions
1. Go to GitHub and create a new public repository.
2. Initialize Git in your `portfolio` folder.
3. Add, commit, and push your code.
4. Go to **Settings > Pages** in your GitHub repo and set the branch to `main`.
5. Wait 1 minute and visit your live URL!

### Submission
Submit the URL to your live website and your GitHub repository link.
