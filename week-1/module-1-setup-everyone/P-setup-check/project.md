# Project: Setup Verification

### Objective
Ensure your environment is ready for coding by generating a "System Check" report.

### Milestone Goals
- Verify VS Code version.
- Verify Git installation.
- Verify Node.js version.

### Instructions
1. Open your Terminal (Integrated in VS Code: `Ctrl + ``).
2. Run the following commands:
   - `code --version`
   - `git --version`
   - `node -v`
3. Take a screenshot of your terminal showing the output of these commands.
4. Save the screenshot as `setup-check.png`.

### Submission
Upload your screenshot to the portal to mark this milestone as complete.
