# Lesson: VS Code & Extensions

### Overview
Visual Studio Code (VS Code) is the most popular code editor in the world. It is lightweight but powerful, allowing you to write code with speed and precision.

### Key Concepts
- **Workspace**: The folder where your project lives.
- **Extensions**: Plugins that add features like auto-completion and code formatting.
- **Settings**: Customizing your editor to fit your flow.
