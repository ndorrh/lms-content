# Project: Portfolio Milestone 2 - Styling

### Objective
Apply CSS styles to the portfolio skeleton you built in the previous module. 

### Milestone Goals
- Create a `style.css` file and link it to `index.html`.
- Add a custom font using Google Fonts.
- Style your header with a background color.
- Add padding and margins to your bio section for better readability.

### Instructions
1. Open your `portfolio` directory.
2. Create `style.css`.
3. In `index.html`, add `<link rel="stylesheet" href="style.css">`.
4. Add the following styles:
   - Center your `<h1>` name.
   - Give the `<body>` a light background color.
   - Use `max-width: 800px` and `margin: 0 auto` on your `<main>` tag.
5. Make a button that changes color when you hover over it!

### Submission
Submit your updated `portfolio` folder or a live link (if you already know how to host it!).
