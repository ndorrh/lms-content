# Lesson: CSS Selectors & the Box Model

### Overview
If HTML is the foundation, CSS is the architecture and interior design. Every element on a webpage is essentially a "box."

### Key Concepts
- **Selectors**: How to target elements.
- **Colors & Typography**: `color`, `background-color`, `font-family`.
- **The Box Model**:
  - `padding`: Space inside the border.
  - `border`: The line around the content.
  - `margin`: Space outside the border.
- **Flexbox Basics**: The modern way to align elements (`display: flex`).
