# Project: The JS Sandbox

### Objective
Verify that your JavaScript is properly connected to your HTML and that you can output information to the console.

### Milestone Goals
- Create a `script.js` file.
- Connect it to a temporary `test.html` file.
- Log a personalized message to the console.

### Instructions
1. Create a new folder named `js-sandbox`.
2. Inside, create `test.html` and `script.js`.
3. In `test.html`, link your script using `<script src="script.js"></script>` just before the closing `</body>` tag.
4. In `script.js`, write a function that logs "Hello from TechAscend!" to the console and then call it.
5. Open `test.html` in your browser and check the console.

### Submission
Add a screenshot of your browser's console to your GitHub repository and submit the repository link.
