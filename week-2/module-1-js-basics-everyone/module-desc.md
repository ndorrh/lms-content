# Module: JavaScript Fundamentals

## Description
Every programming journey starts with the basics. In this module, you will learn the fundamental building blocks of JavaScript. We will cover how to store information using variables, perform different types of data operations, and group logic into reusable functions.

## Learning Objectives
- Declare variables using `let` and `const`.
- Understand basic data types (String, Number, Boolean).
- Create and call your first JavaScript functions.
- Use the Browser Console for debugging.
