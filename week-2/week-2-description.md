# Week 2: Adding Interactivity with JavaScript

## Overview
Now that you have built and styled your personal portfolio, it's time to bring it to life! This week, we introduce JavaScript, the "brain" of the web. You will learn how to make your pages interactive, handle user actions like clicks, and modify your content dynamically without reloading the page.

## Learning Objectives
- Understand the role of JavaScript in web development.
- Learn basic programming concepts: variables, data types, and functions.
- Master DOM selection (`getElementById`, `querySelector`).
- Use event listeners to trigger actions.

## The Milestone: Interactive Portfolio
By the end of this week, your static portfolio will become an interactive experience. You will add a feature that allows users to interact with your page and see immediate feedback.
