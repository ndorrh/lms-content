# Lesson: Selecting & Modifying Elements

### Overview
To change something on a webpage with JavaScript, you must first "find" it. The Browser provides several ways to select elements from the Document Object Model (DOM).

### Key Concepts
- **`document`**: The entry point to the web page content.
- **`getElementById("id")`**: The most efficient way to find a unique element.
- **`querySelector(".class")`**: A flexible way to find elements using CSS selectors.
- **`textContent`**: Getting or setting the text inside an element.
