# Module: The DOM (Document Object Model)

## Description
The DOM is the bridge between your static HTML and your dynamic JavaScript. In this module, you will learn how JavaScript "sees" your webpage as a tree of objects. You will master the techniques to select specific elements by their ID or CSS selectors and change their content instantly.

## Learning Objectives
- Understand the DOM tree structure.
- Select elements using `getElementById` and `querySelector`.
- Manipulate element content using `textContent` and `innerHTML`.
