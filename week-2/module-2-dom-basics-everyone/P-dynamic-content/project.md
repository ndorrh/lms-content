# Project: Portfolio Milestone - Dynamic Greeting

### Objective
Apply your DOM selection skills to your personal portfolio by making the greeting dynamic.

### Milestone Goals
- Add a unique ID to your portfolio's main heading or bio.
- Use JavaScript to change the text of that element when the page loads.

### Instructions
1. Open your `portfolio/index.html` from Week 1.
2. Ensure your `script.js` is linked.
3. Add `id="hero-title"` to your main `<h1>` element.
4. In `script.js`, select that element and change its `textContent` to something like "Dynamic Portfolio of [Your Name]".
5. Refresh the page to see the text change!

Add the screenshot and `script.js` code snippet to your GitHub repository and submit the repository link.
