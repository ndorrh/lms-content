# Project: Portfolio Milestone - The Interactive Button

### Objective
This is the final milestone for Week 2! You will add a functional, interactive element to your personal portfolio using everything you've learned.

### Milestone Goals
- Add a button to your Portfolio's landing page.
- Write an event listener that triggers a change when the button is clicked.

### Instructions
1. In your `portfolio/index.html`, add a `<button id="cta-button">Click to Learn More</button>`.
2. Add a hidden or empty `<div>` or `<p>` with `id="secret-message"`.
3. In `script.js`, select both the button and the message element.
4. Add a `click` event listener to the button.
5. Inside the listener, change the `textContent` of the message element to reveal a fun fact about yourself or a welcome message.
6. **Bonus**: Change the button's background color or text once it has been clicked!
