# Lesson: Event Listeners

### Overview
Events are actions or occurrences that happen in the system you are programming, which the system tells you about so your code can respond to them.

### Key Concepts
- **`addEventListener("name", callback)`**: The standard way to listen for events.
- **Common Events**: `click`, `submit`, `mouseover`, `keydown`.
- **The Event Object**: Information about the event (e.g., which button was clicked).
